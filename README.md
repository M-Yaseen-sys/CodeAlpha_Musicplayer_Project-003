# 🎵 Music Player Web App

A modern and responsive **Music Player Web Application** built using **HTML5, CSS3, and JavaScript**. This project provides a clean and interactive interface that allows users to play music, pause tracks, skip to the next or previous song, adjust the volume, and navigate through a playlist.

The player updates song information dynamically, displays playback progress in real time, and automatically plays the next song when the current track ends. This project was developed to strengthen my understanding of JavaScript, DOM manipulation, audio APIs, and responsive web design.

---

## ✨ Features

* ▶️ Play & Pause music
* ⏮ Previous song
* ⏭ Next song
* 📂 Playlist support
* 🎵 Dynamic song title & artist
* 📊 Real-time progress bar
* ⏱ Current time & duration display
* 🔊 Volume control
* 🔄 Auto-play next track
* 📱 Responsive and clean user interface
